## 📌 프로젝트 개요
**TRip TRip **는 여행자와 숙소 제공자를 연결하는 **여행 커뮤니티 & 커머스 플랫폼**입니다.  

- 여행자: 여행지 소식 공유 & 다른 유저와 소통  
- 숙소 예약: 원하는 숙소를 검색·예약  
- 숙소 제공자: 내 숙소를 등록하고 판매  

---

## ⚙️ 기술 스택
- **Frontend**: React, Next.js, TypeScript
- **Responsive UI**: Desktop, Tablet, Mobile 대응  
- **Core Languages**: JavaScript, HTML, CSS
